# JavaScript Ternary Operator

This repository contains study material and practice exercises on the JavaScript ternary operator. It demonstrates how to write concise conditional expressions for efficient code.

## Concepts Covered
- Basic Ternary Syntax  
- Conditional Expressions  
- Nested Ternary Operators  
- Practical Examples  

## Usage
Run the JavaScript files in the browser console or using Node.js to practice and test examples.
